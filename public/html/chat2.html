¿Como usar el chat?

Para usar el chat lo primero que tendremos que hacer es registrarnos o entrar en la web, una vez hecho esto si volvemos a esta web podremos ver como aparecera un chat donde podremos escribir y ver los mensajes tiempo real con otra gente.

Y como si una aplicacion para enviar mensajes tipo whatsapp o telegram se tratase podras empezar a hablar con quien quieras.

¿Porque tendria que usar este chat y no otro?

Lo primero que nos ofrece este chat que no otros es que veremos la gente cercana a nosotros conecta y si queremos hablar con ellos solo tendremos que pulsar sobre su perfil y ya podremos hablar con ellos/as, como se muestra mas abajo.
